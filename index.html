About This Visualization

This advanced earthquake visualization displays worldwide seismic activity from the USGS Earthquake Hazards Program.

Features

  • Real-time earthquake data for the past 7 days
  • Multiple map views including street, satellite, topographic, and dark mode
  • Tectonic plate boundaries
  • Magnitude and depth filtering
  • Time-based animation
  • Earthquake heatmap visualization
  • Detailed statistics and information

How to Use

  • Use the layer control (top right) to switch between different map views and data layers
  • Apply magnitude and depth filters with the control panel (top left)
  • Use the time slider (bottom left) to see earthquakes as they occurred through time
  • Click on any earthquake for detailed information

Data Sources

All earthquake data comes from the USGS Earthquake Hazards Program.

Tectonic plate boundaries from fraxen/tectonicplates.
